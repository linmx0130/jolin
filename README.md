jolin
===
**J**ust **O**ne **LIN**ear algebra library in Rust. Still in development.

* Focus on simplicity and avoid dependencies on any other libraries.
* Few optimizations.
* **Not recommended for serious use now.** Use BLAS/LAPACK instead.
